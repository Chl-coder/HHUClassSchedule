# HHUClassSchedule

> HHU课程表是一个简洁高效无广告的课程表软件，正在开发中

# 特点

- 简洁，无广告
- 支持手动添加、导入课程表
- 支持新URP教务系统课表的导入
- 纯java开发，适合新手学习，二次开发



# 开发流程

1. fork本项目到自己的仓库
2. 获取自己仓库的git链接，clone到本地开发
3. push到自己的远程仓库
4. pr提交到dev分支



# 注意事项

- pr提交到dev分支，不是mian分支
